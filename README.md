# django-graphql-sample

Graphene-Djangoを使用したAPIサーバのサンプル

## Installation

- Execute `cp .env.sample .env`
- Fill `.env`
- Execute `make up` to run the containers
- Execute `make migrate` to migrate
- Execute `make admin` to create superuser
